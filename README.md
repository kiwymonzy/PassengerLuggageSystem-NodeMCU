PASSENGER LUGGAGE SYSTEM

Introduction
The Passenger Luggage Monitoring System is a project that combines both hardware and software components to monitor and manage luggage in a passenger transportation system. This system aims to enhance security and ensure efficient handling of passenger luggage.

## Hardware Requirements
To run the hardware part of the system, you will need the following:

- Arduino Uno board
- Proteus software for simulation

## Software Requirements
The software part of the system can be accessed using the following link:

- [http://localhost:3000/](http://localhost:3000/)

Default login credentials for the software are as follows:
- Username: admin
- Password: admin

## Installation and Setup

### Hardware Setup
1. Connect your Arduino Uno board to your computer.
2. Launch Proteus software.
3. Open the project file provided for the Passenger Luggage Monitoring System.
4. Simulate the project in Proteus to run the hardware part.

### Software Setup
npm install
npm start

1. Ensure you have a working internet connection.
2. Open a web browser.
3. Go to [http://localhost:3000/](http://localhost:3000/).
4. Enter the default username "admin" and the default password "admin" to log in to the system.
5. Once logged in, you can start using the software part of the Passenger Luggage Monitoring System.

## Features
The Passenger Luggage Monitoring System offers the following features:

1. User Authentication:
   - Log in using the provided credentials (username and password).
   - Secure access to the system.

2. Luggage Monitoring:
   - Real-time monitoring of passenger luggage.
   - Tracking the location and status of luggage.
   - Alert notifications for any suspicious activities or unauthorized access to luggage.

3. Luggage Management:
   - Add new luggage to the system.
   - Edit existing luggage details.
   - Delete luggage from the system.
   - Search for specific luggage based on criteria such as passenger name or luggage ID.

4. Reporting and Analytics:
   - Generate reports on luggage status, history, and statistics.
   - View analytics and insights on luggage handling and transportation efficiency.

## Troubleshooting
If you encounter any issues or difficulties while running or using the Passenger Luggage Monitoring System, please follow these steps:

1. Check your hardware connections and ensure that the Arduino Uno board is properly connected.
2. Verify that you have installed Proteus correctly and the project file is compatible with the software version you are using.
3. For software-related issues, make sure you have a stable internet connection and try refreshing the webpage.
4. If you continue to experience problems, please refer to the user documentation or contact our technical support team for assistance.
